## 🏦 카카오뱅크 데모 페이지

이 프로젝트는 **카카오뱅크 데모 페이지**를 제작한 것입니다.  
HTML, CSS, JavaScript, jQuery, AJAX, Bootstrap을 활용하여 개발하였습니다.

### 📅 프로젝트 기간  
🗓 **2025년 2월 11일 ~ 2월 21일**

### 🚀 사용 기술  
- **Frontend**: HTML, CSS, JavaScript, Bootstrap  
- **JavaScript 라이브러리**: jQuery, AJAX  
- **프로젝트 관리**: Jira  
- **프로토타이핑**: Figma  

### 📌 주요 기능  
- 카카오뱅크 메인 페이지 스타일링  
- 카카오뱅크 주요 페이지 스타일링
- AJAX를 활용한 비동기 데이터 처리  
- jQuery를 사용한 동적 UI 구성
